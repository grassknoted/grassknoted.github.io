Multilingual, end-to-end, AI-based disaster management platform using a custom protocol over GSM (no internet required), with network triangulation. Features: Proprietary communcation protocol using a GSM module, AI-chatbot over GSM, Automated Cloud-based call-center, Smart maps with Google Maps API, Victim and Rescuer dashboard, and one-shot facial recognition for victim identification using Microsoft Azure API.



Technologies Used: Raspberry Pi, Embedded systems, Neural Networks, Deep Learning, Cloud Computing, Microsoft Azure.

My Role:
  • Designed the GSM Communication protocols.
  • Developed the rescuer and victim dashboards.
  • User feedback from Kerala flood victims.
  • Integration with Microsoft Azure APIs.
  • Integration with Google Maps APIs.
