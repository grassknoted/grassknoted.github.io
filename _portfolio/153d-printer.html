Designed and built a 3D printer completely from scratch, complete with the Operating System built over Ramps. The printer takes a 2D image as input and prints out the image in 3-Dimensions, using a combination of Photogrammetry and Computational Imaging.

Technologies Used: Hardware, Design, Ramps 1.4, Raspberry Pi, Cura, Slic3r.

My Role:
  • Designed the parts for the 3D printer.
  • Built the operation system for the printer over Ramps 1.4.
  • Optimized all parts for a 800cc build volume.
  • Create a pipeline involving Slic3r and Cura to create 3D templates from 2D images.
